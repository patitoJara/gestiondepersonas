Seguridad y Sesión

info
El sistema cuenta con mecanismos de seguridad orientados a proteger la información clínica y administrativa, garantizando confidencialidad, integridad y disponibilidad de los datos conforme a las normativas institucionales del Servicio de Salud Magallanes.

Las principales medidas de seguridad implementadas son:

  • Autenticación mediante usuario y contraseña institucional.
  • Generación de token seguro de sesión.
  • Control de sesión con expiración automática.
  • Renovación manual antes del vencimiento.
  • Control de acceso basado en roles y programas.
  • Registro seguro de operaciones.

1. Ingreso al Sistema

Para acceder al sistema, el usuario debe ingresar su correo electrónico institucional y contraseña en la pantalla de inicio de sesión.

  • Ingrese su correo electrónico institucional.
  • Ingrese su contraseña.
  • Presione el botón Iniciar sesión.
warning
Después de 60 minutos de inactividad, la sesión expirará automáticamente para proteger la información del usuario.

2. Recuperar Contraseña

Si el usuario ha olvidado su contraseña, puede utilizar la opción “Recuperar contraseña” disponible en la pantalla de inicio.

  • Ingrese su correo electrónico institucional.
  • Presione Enviar correo.
  • El sistema enviará una notificación al área de soporte con copia al usuario.

3. Cambio de Contraseña (desde recuperación)

Una vez autorizado el cambio, el usuario podrá ingresar una nueva contraseña desde la pantalla habilitada para tal efecto.

  • Ingrese la nueva contraseña.
  • Confirme la nueva contraseña.
  • Presione Actualizar.
info
La contraseña debe tener al menos 8 caracteres y se recomienda combinar letras, números y símbolos para mayor seguridad.

4. Cambio de Contraseña desde Mi Perfil

El usuario puede cambiar su contraseña en cualquier momento desde el menú superior del sistema.

  • Seleccione su nombre en la barra superior.
  • Ingrese a Mi Perfil.
  • Complete los campos solicitados.
  • Presione Actualizar contraseña.

5. Renovación de Sesión

Cuando la sesión esté próxima a expirar, el sistema mostrará un contador visible en la barra superior indicando el tiempo restante.

  • El contador muestra los minutos restantes de sesión.
  • Antes de expirar, el usuario podrá extender la sesión manualmente.
  • Al renovar la sesión, el sistema genera un nuevo token de seguridad.
schedule
La renovación evita la pérdida de información en formularios que se encuentren en proceso.
