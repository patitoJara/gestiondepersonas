5. Administración de Usuarios

Esta sección permite la gestión de cuentas de acceso al sistema de registro de eventos de teletrabajo.

Gestión de Cuentas

  • Crear usuarios.
  • Editar información de usuarios.
  • Activar o desactivar cuentas.

Gestión de Roles

  • Asignar roles (Funcionario, Administrador, Supervisor).
  • Modificar roles existentes.
