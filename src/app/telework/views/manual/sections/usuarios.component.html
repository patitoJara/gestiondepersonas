5. Administración de Usuarios

Esta sección permite la gestión de cuentas de acceso al sistema de registro de eventos de teletrabajo.

Gestión de Cuentas

  • Crear usuarios.
  • Editar información de usuarios.
  • Activar o desactivar cuentas.

Gestión de Roles

  • Asignar roles (Funcionario, Administrador, Supervisor).
  • Modificar roles existentes.
verified_user
Solo los usuarios con rol Administrador pueden gestionar cuentas y asignaciones de roles dentro del sistema.
person_search
Selección de Rol en Inicio de Sesión:

Cuando un usuario posee múltiples roles asignados, el sistema solicitará seleccionar el rol activo al momento de ingresar.

El rol seleccionado determina:
- Los accesos disponibles dentro del sistema.
- Las opciones visibles en el menú lateral.
- Las funcionalidades habilitadas para el usuario.

El rol activo se mantiene durante toda la sesión del usuario y puede influir directamente en la visibilidad y operación de los módulos del sistema.
